MobxApp

Class 属性及方法：继承 BaseApp

  • store

    即通过 appify() 函数注入的 store 对象

Example：


import {MobxStore, MobxApp, appify} from '@minapp/mobx'

@appify(
  new MobxStore(),
  {
    pages: require('./app.json?pages'),
    tabBarList: require('./app.json?tabBar.list')
  }
)
export default class extends MobxApp {
  async onLaunch() {
    // todo
  }
}
